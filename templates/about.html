ABOUT SPAMSHIELD

An AI-powered email security platform using Machine Learning to detect spam, analyse content, and protect users from phishing in real time.

⚙️ How It Works

  • 1
    Email InputUser pastes or types the email content into the detection panel.
  • 2
    Text VectorizationEmail is transformed into a numerical feature vector using Bag of Words (CountVectorizer).
  • 3
    Naive Bayes ClassificationMultinomial Naive Bayes predicts spam or legitimate based on word probabilities.
  • 4
    Deep NLP AnalysisAdditional checks — keyword detection, emotion analysis, trust scoring, difficulty rating, and category classification.
  • 5
    Result & ReportFull visual report with confidence score, probability bar, AI badges, and a downloadable PDF.

🤖 AI Features

  • Naive Bayes spam classification
  • Spam probability bar (visual AI result)
  • AI confidence badge — HIGH / MEDIUM / LOW
  • Spam difficulty — Easy / Medium / Advanced
  • Email emotion detection (Urgent, Threatening…)
  • Category classifier (Spam, Work, Personal, Promotion)
  • Trust score 0–100 (Spam / Suspicious / Safe)

🔍 Detection Signals

  • 25+ spam keyword detection with highlighting
  • External URL / link extraction
  • Exclamation mark count analysis
  • Word count and character count
  • Phishing pattern recognition
  • Risk signals panel per email
  • Spam warning popup alert

📊 Dashboard & Analytics

  • Chart.js doughnut — Spam vs Ham
  • Confusion matrix from test set
  • Time-based tracker (Today vs Yesterday)
  • 7-day spam trend line chart
  • Emotion breakdown bar chart
  • Category breakdown bar chart
  • Average trust score across emails

🛠️ System Features

  • Session-based login with logout
  • Light & Dark theme toggle
  • Full email history with search & filter
  • Copy email content & result to clipboard
  • Downloadable styled PDF report
  • Sample email chips for quick testing
  • Animated UI with real-time feedback

🧠 Machine Learning Stack

  • Algorithm: Multinomial Naive Bayes
  • Vectorizer: Bag of Words (CountVectorizer)
  • Library: scikit-learn
  • Dataset: SMS Spam Collection (spam.csv)
  • Train/Test Split: 80% / 20%
  • Model persistence: pickle (.pkl files)

💻 Tech Stack

  • Backend: Python + Flask
  • Frontend: HTML5, CSS3, Vanilla JS
  • Charts: Chart.js v4
  • PDF Export: ReportLab
  • Fonts: Orbitron, Rajdhani, JetBrains Mono
  • Themes: CSS variables (Dark + Light)
